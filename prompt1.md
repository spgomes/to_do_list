Crie uma aplicação web em Golang de lista de tarefas (TO DO) simples. A aplicação deve permitir ao usuário:

Adicionar novas tarefas
Listar todas as tarefas cadastradas
Marcar tarefas como concluídas
Remover tarefas da lista

A interface deve ser intuitiva e responsiva, com campos para inserir o nome da tarefa e botões para as ações (adicionar, concluir, remover). Utilize boas práticas de desenvolvimento e organize o código de forma clara. Se possível, implemente persistência local (localStorage) para manter as tarefas salvas entre sessões.
